# WDColorPicker
WDColorPicker is a simple lightweight component for displaying color picker inside iOS apps.

# Installation:
## CocoaPods:
```Ruby
target '<TargetName>' do
    use_frameworks!
    pod 'WDColorPicker', ' ~> 0.0.5'
end
```
